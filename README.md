# How to use
克隆仓库到本地

一定要记得修改根目录为名PythonPlugins

打开cmd运行如下命令

   pip install -r requirements.txt

修改并运行bot.py(修改QQ为你bot的QQ,port、key与你的mirai-api-http需要保持一致)



发送语音指令如下：
-


发送命令（私聊窗口用）

      发送×××

中文生成（给群友用）

      #中文
日语生成（给群友用）

      #说

向群聊发送（私聊用群号为示例）

       连接群#699455559
向个人发送（由于腾讯更新私聊语音加密，不能用了）

        连接对象#1840094972

语言切换

         切换中文
         切换日语
 
模型切换

          M#0
          M#1
          M#2
          M#3
